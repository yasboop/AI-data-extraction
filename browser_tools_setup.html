BrowserTools MCP Setup Guide

This guide will help you set up the AgentDesk BrowserTools MCP for use with Cursor.

Good news! You've already completed the following steps:

  • mcp.json configuration has been updated with the correct path to npx
  • The BrowserTools server is running on port 3025
  • The BrowserTools MCP is running and will be detected by Cursor

Step 1: Download the Chrome Extension

The Chrome extension isn't on the Chrome Web Store yet, so you'll need to download it from GitHub:

  1. Go to: https://github.com/AgentDeskAI/browser-tools-mcp/releases/tag/v1.2.0
  2. Download the file named BrowserToolsMCP-ChromeExtension-v1.2.0.zip
  3. Extract the ZIP file to a folder on your computer

Step 2: Install the Chrome Extension

To install the downloaded extension:

  1. Open Chrome
  2. Navigate to chrome://extensions/
  3. Enable "Developer mode" using the toggle in the top right
  4. Click "Load unpacked"
  5. Select the folder where you extracted the extension
  6. You should see "BrowserToolsMCP" appear in your extensions list

Step 3: Restart Cursor

For the mcp.json changes to take effect:

  1. Close Cursor completely
  2. Reopen Cursor
  3. Start a new chat session

Step 4: Test the Browser Tools

To verify that everything is working:

  1. Open any website in Chrome
  2. Open Chrome DevTools (Right-click â†’ Inspect)
  3. Look for the "BrowserTools" tab in DevTools
  4. In Cursor, start a new chat and try commands like:
    Please check the browser logs
    Take a screenshot of the current page
    Check for any console errors

Note: If you encounter any issues, try the following troubleshooting steps:

  • Ensure both the server and MCP are running (you can restart them using npx @agentdeskai/browser-tools-server@1.2.0 and Cursor restart)
  • Check that Chrome DevTools is open when you're using the Browser Tools
  • Make sure you've restarted Cursor after updating the mcp.json file
